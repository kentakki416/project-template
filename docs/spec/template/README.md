# specのテンプレート
spec/templateのテンプレートです。templateは1-authenticateや2-createTodoなど機能単位でディレクトリが作成されます

## 仕様
機能の使用を箇条書きでできるだけ詳細に記述してください

## 必要なAPI
テーブル形式でAPIと説明を追加してください

## 必要な画面
必要となる画面を説明してください

## 必要なDB設計
必要なDBをテーブルとリレーションをテーブル形式で簡潔に記載

## フロー図
APIリクエストの流れをシーケンス図をmermaidで記述

## 注意事項
セキュリティやパフォーマンス、非機能要件などを記述してください。
