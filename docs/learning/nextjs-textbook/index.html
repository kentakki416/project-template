Next.js 実践教科書

既存の apps/web を土台に、web 版「メモ機能」を新しいページとして章ごとに積み上げながら App Router のベストプラクティスを体得する build-along 教材。

この本のねらい

参考書で一度読んだ知識を、実際に動くコードを自分の手で足していくことで「使える」レベルに引き上げます。各章は「概念 → 作るコード → 逐行解説 → 図解 → なぜこう書くか → ハンズオン＆クイズ」で構成しています。

前提と進め方

  • 対象: Next.js を書籍レベルで一度学んだ人。React の基本（JSX / props / hooks）は既知とします。
  • 教材の題材: この monorepo には memo の API（CRUD 完備）・スキーマ・mobile 実装が既にありますが、web にはまだメモ機能がありません。これを一から作ります。
  • この本の約束ごと: 既存 web のコードは改修しません。あなたが apps/web/src に新しいファイルを足していくのが学習の本体です。この HTML はその手引きです。
  • 動かし方: pnpm dev で web を起動し、ログインは /api/dev/login?as=alice（開発専用）を使います。詳しくは第0章。
📌 バージョンについて

本書は next@16 / react@19 を前提にしています。Next.js はキャッシュや API のデフォルトがメジャーバージョンで変わるため、細部は必ず公式ドキュメントの当該バージョンで確認してください。本文でも版依存の箇所には「互換メモ」を添えています。
